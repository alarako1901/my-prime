# My Prime — solo 66 días mas

App de hábitos basada en el principio de los 66 días para consolidar un hábito.

## Cómo funciona

- Marca tus hábitos cada día. Solo el **día actual** es editable; los días pasados quedan bloqueados como registro.
- **PRIME** por hábito: porcentaje de avance sobre los 66 días totales.
- **Niveles de desbloqueo**: 7 → 15 → 30 → 66 días.
- **PROGRESO**: gráfico diario del % de hábitos cumplidos.
- **PRIME TOTAL**: tu avance global en el reto.

## Ver la app

Abrir `index.html` en el navegador, o visitar la versión publicada con GitHub Pages.

## Stack

HTML + CSS + JavaScript puro (prototipo). Próximo paso: React + Vite, luego React Native + Expo para iOS.
